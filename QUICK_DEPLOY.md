## Step-by-Step Deployment

### 1. Go to Vercel
Open: https://vercel.com/login

### 2. Sign In / Sign Up
- If you have an account: Sign in with GitHub (recommended)
- If you don't have an account: Click "Sign Up" and use your GitHub account

### 3. Import Your Project
1. After logging in, click **"Add New Project"** button
2. You'll see a list of your GitHub repositories
3. Find and select: **`ashwagandha`** (or `somayshsadani/ashwagandha`)
4. Click **"Import"**

### 4. Configure (Auto-detected - Just Verify)
Vercel will auto-detect your static site. Verify these settings:
- **Framework Preset**: `Other` (or it might say "No Framework")
- **Root Directory**: `./` (leave as default)
- **Build Command**: Leave **EMPTY** (no build needed)
- **Output Directory**: Leave **EMPTY** (or `./`)
- **Install Command**: Leave **EMPTY**

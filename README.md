# Biolve - Bulk Ashwagandha Powder Supplier Website

A static, compliance-first B2B website for bulk Ashwagandha (Withania somnifera) root powder supply, targeting US dietary supplement manufacturers and distributors.

## Features

- **Static HTML/CSS only** - No frameworks, no JavaScript, no animations
- **Compliance-first language** - Conservative, procurement-friendly tone with no health or functional claims
- **Professional B2B design** - White background, subtle grey accents, clean typography
- **Complete site structure** - Home, Product, Quality & Compliance, About, and Contact pages

## Pages

- `index.html` - Homepage with product overview and procurement information
- `product.html` - Detailed product specifications and commercial details
- `quality.html` - Quality & compliance documentation for procurement teams
- `about.html` - Company overview focused on sourcing and export experience
- `contact.html` - Inquiry form and procurement contact information

## Design

- Minimal, professional aesthetic
- Responsive layout
- Accessible markup
- Conservative color palette (white, greys)

## Compliance

This website does not provide medical advice and does not make health or functional claims. All product decisions and labeling are the responsibility of the manufacturer and/or brand owner.

## Deployment

This site can be easily deployed to Vercel. See [DEPLOYMENT.md](./DEPLOYMENT.md) for detailed instructions.

**Quick deploy to Vercel:**
```bash
npm i -g vercel
vercel login
vercel --prod
```

Or connect your Git repository to Vercel for automatic deployments.
